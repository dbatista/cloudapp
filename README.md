# Cloudapp

## Aplicação para demonstração de deploy no IBM Cloud Foundry services

### Para executar a aplicação localmente
```
git clone https://github.com/dbatista/cloudapp.git
```
```
npm install
```
```
npm start
```
```
curl --location --request GET 'localhost:3000/'
```
```
Resultado:
{"result":"Working"}
```
